# Contributing to SyncNPlay

Thanks for your interest in contributing!

## Getting Started

1. Fork the repo
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/syncnplay.git`
3. Create a branch: `git checkout -b feat/your-feature`
4. Make your changes
5. Commit using conventional commits: `feat:`, `fix:`, `docs:`, `chore:`
6. Push and open a Pull Request

## Good First Issues

Check issues tagged `good first issue` — these are beginner-friendly tasks.

## Areas to Contribute

- Adding support for a new streaming platform (Netflix, Prime Video, Hotstar)
- Improving the popup UI
- Writing tests
- Improving documentation

## Code Style

- Use clear variable names
- Comment non-obvious logic
- Keep functions small and focused
